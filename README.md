# Damian Seals Portfolio

A custom-built portfolio created to showcase my filmmaking work that also demonstrates my ability to design and develop a modern website from the ground up. The site serves as both a creative portfolio and a technical project, combining a curated presentation of my film work with a responsive, media-focused user experience designed and implemented from scratch.

Rather than relying on a template or site builder, every aspect of the project was built by hand using Next.js, TypeScript, and Tailwind CSS.

## Features

* Responsive design for desktop and mobile devices
* Dynamic project pages using Next.js App Router
* Embedded video playback with custom thumbnail overlays
* Curated project gallery and featured work
* Dedicated About page and site-wide contact footer
* Copy-to-clipboard contact functionality
* Custom styling built with Tailwind CSS

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS

## Design Goals

* Prioritize visual storytelling
* Create a clean, cinematic presentation
* Keep navigation simple and unobtrusive
* Ensure media-heavy pages remain responsive and performant

## Challenges

One of the more interesting challenges was creating a smooth video playback experience across desktop and mobile devices. Mobile browsers handle embedded video differently than desktop browsers, requiring custom logic to preserve a seamless transition from thumbnail to playback while maintaining intuitive user interaction.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site locally.
